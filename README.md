# Описание
---
Android приложение на Kotlin, отслеживающая отключения связи в Москве. 

# Принцип работы
---
Приложение работает на фоне после первого запуска и регулярно отправляет запросы на центральный сервер. Если связь пропадает, то приложение записывают текущую геолокацию как offline. 

Сервер ожидает запроса от конкретного пользователя, и если запросы перестают поступать, он записывает последнюю геолокацию пользователя, как потенциальную offline зону.  

Для корректного отображения на карте online и offline точки кластеризуются, и зоны отображаются на карте. 

Ссылка на бэкенд: https://github.com/artteam8/whitelist_backend
